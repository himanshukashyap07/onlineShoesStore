# **App Name**: SoleSculpt

## Core Features:

- 3D Shoe Showcase: Interactive 3D models of shoes with zoom and rotate functionality for detailed viewing.
- Animated Transitions: Smooth page transitions and UI element animations to enhance user experience.
- Homepage Gallery: Visually engaging homepage gallery featuring the latest shoe models.
- Product Filtering: Dynamic filters (style, size, color, brand) with real-time updates of displayed models.
- AI Style Advisor: AI tool analyzes user preferences and suggests matching shoe models.
- Secure Checkout: Standard checkout process for completing the purchase

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), conveying sophistication and style.
- Background color: Soft lavender (#E6E6FA), creating a calming backdrop that complements the primary color without overwhelming the visual space.
- Accent color: Electric violet (#8F00FF), providing contrast and highlighting interactive elements such as call-to-action buttons and selected product details.
- Body text: 'Inter' sans-serif font, with a modern and readable appearance; Headline text: 'Belleza' sans-serif, which injects personality; the fonts pair well together, balancing legibility and flair.
- Clean, minimalist icons for categories and interactive elements.
- Grid-based layout for a clean and organized presentation.
- Subtle animations for loading, transitions, and interactions.
